Giriş

Akademik makalelerde Table 1 (demografik özellikler tablosu) neredeyse her çalışmada bulunur. Bu tabloyu elle oluşturmak hem zaman alıcı hem de hata yapma riski yüksektir.

R’daki gtsummary paketi, yayına hazır tabloları birkaç satır kodla oluşturmanızı sağlar.

Gerekli Paketler

Kodu Göster 
library(gtsummary)
library(dplyr)

Örnek Veri

Simüle edilmiş bir klinik çalışma verisi oluşturalım:

Kodu Göster 
set.seed(42)
n <- 200

df <- data.frame(
  age = round(c(rnorm(100, 55, 12), rnorm(100, 58, 10))),
  gender = sample(c("Erkek", "Kadın"), n, replace = TRUE, prob = c(0.55, 0.45)),
  bmi = round(c(rnorm(100, 26, 4), rnorm(100, 28, 5)), 1),
  smoking = sample(c("Evet", "Hayır"), n, replace = TRUE, prob = c(0.3, 0.7)),
  diabetes = sample(c("Var", "Yok"), n, replace = TRUE, prob = c(0.25, 0.75)),
  systolic_bp = round(c(rnorm(100, 128, 15), rnorm(100, 135, 18))),
  treatment = rep(c("Plasebo", "Tedavi"), each = 100)
)

Table 1: Demografik Tablo

Kodu Göster 
df |>
  select(age, gender, bmi, smoking, diabetes, systolic_bp, treatment) |>
  tbl_summary(
    by = treatment,
    label = list(
      age ~ "Yaş",
      gender ~ "Cinsiyet",
      bmi ~ "BMI (kg/m²)",
      smoking ~ "Sigara",
      diabetes ~ "Diyabet",
      systolic_bp ~ "Sistolik KB (mmHg)"
    ),
    statistic = list(
      all_continuous() ~ "{mean} ({sd})",
      all_categorical() ~ "{n} ({p}%)"
    ),
    digits = list(
      all_continuous() ~ 1,
      all_categorical() ~ c(0, 1)
    )
  ) |>
  add_p() |>
  add_overall() |>
  modify_header(label = "**Değişken**") |>
  modify_spanning_header(c("stat_1", "stat_2") ~ "**Tedavi Grubu**") |>
  bold_labels()

Tabloyu Dışa Aktarma

Kodu Göster 
# Word formatında kaydetme
tbl <- df |>
  tbl_summary(by = treatment) |>
  add_p()

# Word
tbl |> as_flex_table() |> flextable::save_as_docx(path = "table1.docx")

# HTML
tbl |> as_gt() |> gt::gtsave("table1.html")

# LaTeX
tbl |> as_kable_extra(format = "latex")

İpuçları

  1. add_p(): Otomatik olarak uygun istatistiksel testi seçer (t-test, Wilcoxon, chi-square, Fisher)
  2. add_overall(): Toplam sütunu ekler
  3. bold_p(): Anlamlı p değerlerini kalın yapar
  4. modify_footnote(): Dipnot ekleyebilirsiniz
  5. Dergi formatı: Çoğu dergi Word formatında tablo kabul eder - as_flex_table() kullanın

## Giriş

Akademik makalelerde **Table 1** (demografik özellikler tablosu) neredeyse her çalışmada bulunur. Bu tabloyu elle oluşturmak hem zaman alıcı hem de hata yapma riski yüksektir.

R'daki **gtsummary** paketi, yayına hazır tabloları birkaç satır kodla oluşturmanızı sağlar.

## Gerekli Paketler

```{r}
#| message: false
#| warning: false
library(gtsummary)
library(dplyr)
```

## Örnek Veri

Simüle edilmiş bir klinik çalışma verisi oluşturalım:

```{r}
set.seed(42)
n <- 200

df <- data.frame(
  age = round(c(rnorm(100, 55, 12), rnorm(100, 58, 10))),
  gender = sample(c("Erkek", "Kadın"), n, replace = TRUE, prob = c(0.55, 0.45)),
  bmi = round(c(rnorm(100, 26, 4), rnorm(100, 28, 5)), 1),
  smoking = sample(c("Evet", "Hayır"), n, replace = TRUE, prob = c(0.3, 0.7)),
  diabetes = sample(c("Var", "Yok"), n, replace = TRUE, prob = c(0.25, 0.75)),
  systolic_bp = round(c(rnorm(100, 128, 15), rnorm(100, 135, 18))),
  treatment = rep(c("Plasebo", "Tedavi"), each = 100)
)
```

## Table 1: Demografik Tablo

```{r}
df |>
  select(age, gender, bmi, smoking, diabetes, systolic_bp, treatment) |>
  tbl_summary(
    by = treatment,
    label = list(
      age ~ "Yaş",
      gender ~ "Cinsiyet",
      bmi ~ "BMI (kg/m²)",
      smoking ~ "Sigara",
      diabetes ~ "Diyabet",
      systolic_bp ~ "Sistolik KB (mmHg)"
    ),
    statistic = list(
      all_continuous() ~ "{mean} ({sd})",
      all_categorical() ~ "{n} ({p}%)"
    ),
    digits = list(
      all_continuous() ~ 1,
      all_categorical() ~ c(0, 1)
    )
  ) |>
  add_p() |>
  add_overall() |>
  modify_header(label = "**Değişken**") |>
  modify_spanning_header(c("stat_1", "stat_2") ~ "**Tedavi Grubu**") |>
  bold_labels()
```

## Regresyon Tablosu

Lojistik regresyon sonuçlarını tablo halinde sunalım:

```{r}
df$outcome <- rbinom(n, 1, prob = 0.3 + 0.01 * (df$age - 50) + 0.05 * (df$treatment == "Tedavi"))

model <- glm(outcome ~ age + gender + bmi + smoking + treatment,
             data = df, family = binomial)

model |>
  tbl_regression(
    exponentiate = TRUE,
    label = list(
      age ~ "Yaş",
      gender ~ "Cinsiyet",
      bmi ~ "BMI",
      smoking ~ "Sigara",
      treatment ~ "Tedavi Grubu"
    )
  ) |>
  add_global_p() |>
  bold_p() |>
  bold_labels()
```

library(survival)

## İpuçları

1. **`add_p()`**: Otomatik olarak uygun istatistiksel testi seçer (t-test, Wilcoxon, chi-square, Fisher)
2. **`add_overall()`**: Toplam sütunu ekler
3. **`bold_p()`**: Anlamlı p değerlerini kalın yapar
4. **`modify_footnote()`**: Dipnot ekleyebilirsiniz
5. **Dergi formatı**: Çoğu dergi Word formatında tablo kabul eder - `as_flex_table()` kullanın
