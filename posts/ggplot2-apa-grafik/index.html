Giriş

Akademik makalelerde grafikler, bulguların etkili sunumu için kritik öneme sahiptir. APA (American Psychological Association) formatı, birçok sosyal bilim ve sağlık bilimleri dergisinde standart olarak kullanılır.

Bu yazıda ggplot2 ile APA standardına uygun grafikler oluşturacağız.

APA Grafik Kuralları

  • Arka plan: Beyaz (grid çizgileri minimumda)
  • Font: Sans-serif (Arial, Helvetica)
  • Renk: Siyah-beyaz baskıda da anlaşılır olmalı
  • Eksen etiketleri: Açık ve anlaşılır
  • Başlık: Figür numarası ile (Figure 1, Figure 2…)

Temel APA Teması

Kodu Göster 
library(ggplot2)
Warning: package 'ggplot2' was built under R version 4.5.2
Kodu Göster 
library(dplyr)

# APA teması
theme_apa <- function(base_size = 12) {
  theme_classic(base_size = base_size) %+replace%
    theme(
      text = element_text(family = "sans"),
      plot.title = element_text(face = "bold", size = base_size + 2,
                                hjust = 0, margin = margin(b = 10)),
      axis.title = element_text(face = "bold"),
      axis.text = element_text(color = "black"),
      legend.position = "bottom",
      legend.title = element_text(face = "bold"),
      panel.grid = element_blank(),
      strip.background = element_blank(),
      strip.text = element_text(face = "bold")
    )
}

Örnek 1: Grup Karşılaştırma (Bar Chart)

Kodu Göster 
set.seed(42)
df_bar <- data.frame(
  group = rep(c("Kontrol", "Tedavi A", "Tedavi B"), each = 30),
  score = c(rnorm(30, 50, 10), rnorm(30, 62, 12), rnorm(30, 58, 11))
)

summary_df <- df_bar |>
  group_by(group) |>
  summarise(
    mean_score = mean(score),
    se = sd(score) / sqrt(n()),
    .groups = "drop"
  )

ggplot(summary_df, aes(x = group, y = mean_score, fill = group)) +
  geom_col(width = 0.6, color = "black", linewidth = 0.3) +
  geom_errorbar(aes(ymin = mean_score - se, ymax = mean_score + se),
                width = 0.15, linewidth = 0.5) +
  scale_fill_manual(values = c("gray80", "gray50", "gray30")) +
  labs(x = "Tedavi Grubu", y = "Ortalama Skor (±SE)") +
  theme_apa() +
  theme(legend.position = "none") +
  coord_cartesian(ylim = c(0, 80))

Örnek 3: Scatter Plot + Regresyon

Kodu Göster 
set.seed(42)
df_scatter <- data.frame(
  study_hours = runif(80, 1, 10),
  exam_score = NA
)
df_scatter$exam_score <- 35 + 5.5 * df_scatter$study_hours + rnorm(80, 0, 8)

model <- lm(exam_score ~ study_hours, data = df_scatter)
r2 <- summary(model)$r.squared

ggplot(df_scatter, aes(x = study_hours, y = exam_score)) +
  geom_point(shape = 21, fill = "gray60", size = 2.5, stroke = 0.5) +
  geom_smooth(method = "lm", color = "black", fill = "gray85", linewidth = 0.8) +
  annotate("text", x = 2, y = 90,
           label = paste0("R² = ", round(r2, 3)),
           fontface = "italic", size = 4.5) +
  labs(x = "Günlük Çalışma Saati", y = "Sınav Puanı") +
  theme_apa()
`geom_smooth()` using formula = 'y ~ x'
